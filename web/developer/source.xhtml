Use the Source!

K-3D is Free Software. The source code is licensed using the GNU General Public License.

K-3D is written in C++ with platform independence in mind. As a result, K-3D can be built on GNU/Linux, MacOS X, Microsoft Windows, and other platforms.

We use Mercurial for source code management.

We use CMake for the K-3D build system.

K-3D uses gtkmm for its graphical user interface.

Learn more about the source code here.

Quick Start

Clone the K-3D sources and build: 

$ hg clone http://k3d.hg.sf.net/hgweb/k3d/k3d
$ mkdir k3d-build
$ cd k3d-build
$ cmake ../k3d
$ make

Make changes to the K-3D sources: 

$ cd k3d
$ (edit files)
$ hg add (new files)
$ hg rm (files to be deleted)
$ hg commit -m 'My changes'

Create a patch from the most recent commit: 

$ cd k3d
$ hg export tip > my.patch

For more on using Mercurial see the guide.
